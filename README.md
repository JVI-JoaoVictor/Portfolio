# 💻 Portfólio Pessoal

Site de portfólio pessoal desenvolvido para apresentar minhas habilidades, projetos e trajetória como desenvolvedor.

## 🎯 Objetivo

Criar uma vitrine online para centralizar meus projetos, experiências e formas de contato, servindo como cartão de visitas digital para recrutadores e clientes.

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript

## ⚙️ Como rodar localmente

```bash
git clone https://github.com/JVI-JoaoVictor/Portfolio.git
cd Portfolio
```

Depois é só abrir o arquivo `index.html` no navegador (ou usar a extensão Live Server do VS Code).

## 💡 Desafios e aprendizados

- Estruturação de um layout responsivo do zero, sem frameworks CSS
- Organização de código HTML/CSS/JS em um projeto de porte real, pensando em manutenção
- Boas práticas de semântica HTML para acessibilidade e SEO
